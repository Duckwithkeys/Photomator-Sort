DUCKSORT FUTURE ARCHITECTURAL & FEATURE ROADMAP SUGGESTIONS

================================================================================
1. ON-DEVICE AI VISION & MACHINE LEARNING (APPLE SILICON NEURAL ENGINE)
================================================================================

1A. On-Device Face Clustering & Subject Grouping
  - Integrate Apple's VNGeneratePersonSegmentationRequest and VNIdentifyJunkCharactersRequest 
    to automatically group photos by recognized individuals across folders.
  - Maintain a local, private face database in App Support without sending any telemetry 
    or image data to external cloud services.

1B. Automated Aesthetic & Sharpness Quality Scoring
  - Compute Laplacian variance and Vision aesthetic quality scores on input images.
  - Provide a visual "Sharpness Index" badge on thumbnails in the grid so photographers 
    can immediately spot soft focus or motion blur when culling high-speed sports/wildlife shots.

1C. Natural Language Semantic Image Search (CoreML Embeddings)
  - Implement zero-shot text-to-image semantic search using local CLIP or MobileCLIP 
    vector embeddings.
  - Allow users to query their entire photo library with natural language phrases like 
    "red sports car", "golden hour portrait", or "dog playing in snow".

================================================================================
2. PROFESSIONAL WORKFLOW & FILE MANAGEMENT ENHANCEMENTS
================================================================================

2A. Dual-Monitor & Reference Studio Mode
  - Allow detaching the Large Image Viewer into a dedicated secondary window.
  - Photographers can keep the full grid view active on their primary display while 
    projecting color-calibrated full-canvas previews onto a reference monitor.

2B. Batch Photomator / Lightroom Sidecar Exporter
  - Enhance export routing rules to generate standardized sidecars (.photo-edit / .xmp) 
    with embedded color profiles and preset metadata applied across thousands of selected photos in bulk.

2C. Automated NAS & Cloud Backup Sync Engine
  - Add background synchronization profiles for export routes targeting local NAS servers 
    (SMB/NFS) or cloud storage targets with automated checksum verification.

================================================================================
3. NEXT-GEN GRAPHICS & SCALING ARCHITECTURE (PHASE 2 PERFORMANCE)
================================================================================

3A. Ahead-of-Viewport Velocity-Aware Prefetch Coordinator
  - Calculate scroll velocity vectors in ScrollStateObserver to predict exactly which 
    thumbnails will enter the viewport 2-3 screens ahead, pre-warming memory caches before cells appear.

3B. Direct Metal Texture Atlas for Multi-Thousand Photo Grids
  - Render grid thumbnails directly via a unified Metal texture array on Apple Silicon 
    unified memory, completely bypassing main actor SwiftUI view tree allocations for 100,000+ photo libraries.
